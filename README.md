# My project Name is "Wanderlust" Same as Airbnb Website

A fully functional Airbnb clone built using the Full Stack (MongoDB, Express, Node.js). This web application allows users to browse, create, and  listings, similar to Airbnb's core features.

## Features
- **User Authentication**: Sign up, login, and logout functionalities Using Passport.js
-  **Listings**: Users can browse available listings with details such as price, location, image and description.
-  **Admin Panel**: Admins can manage listings, users.
-  **Profile Management**: Users can view and update their profile information.
-  **User interface**: You can write review and rating any property.



## Technologies Used

- **MongoDB**: Database for storing user information, listings , rating and review.
- **Express.js**: Web framework for Node.js to handle server-side logic and routing.
- **Node.js**: JavaScript runtime to run the server-side code.
- **passport**: using login and signup.
- **MongoDB Atlas**: for colud data store
